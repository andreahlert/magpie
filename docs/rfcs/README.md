## Conventions

- New RFC numbers are allocated sequentially within their prefix.
- Prefix tracks lineage: `RFC-AI` for the original Agent Infrastructure series, `RFC-MAG` for Magpie-specific framework design.
- Status values: Draft, In review, Accepted, Superseded, Rejected.
- Every RFC links its tracking issue and any RFC it depends on, in the header table.
